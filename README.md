# Ubuntu i3 wm using Regolith

```
chmod a+x script.sh
./script.sh
```

It configures Ubuntu to use a very friendly and customisable i3wm enviorment.

Also sets up Nano,enables snaps and installs:

- [x] Allacritty terminal
- [x] VS Code
- [x] git
- [x] cmake
- [x] Sublime
- [x] Spotify
- [x] VLC
- [x] Transmission
- [x] screenfetch
- [x] wine and winetricks
